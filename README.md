# Next JS

## Dynamic routing 
[slug] act as a dynamic segamant catcher in url.

e.x :

/blog/post1 --> post1 is dynamic
/blog/post2 --> post2 is dynamic

## Routes Grouping

(auth) used to group the routes under a one folder. Routing system will exclude the grouping folder.

e.x : 
/auth/login ---> wrong 
/login ---> correct
